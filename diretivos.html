Diretivos Descrição
.class Usada para declarar uma classe ou interface.
.super Usada para especificar a superclasse de uma classe.
.implements Usada para especificar as interfaces implementadas por uma classe.
.field Usada para declarar um campo (variável) em uma classe.
.subannotation Usada para especificar uma subanotação dentro de uma anotação.
.annotation Usada para declarar uma anotação.
.enum Usada para declarar um tipo enum.
.method Usada para declarar um método em uma classe.
.registers Usada para especificar o número máximo de registradores necessários para um método.
.locals Usada para especificar o número máximo de variáveis locais necessárias para um método.
.array-data Usada para inicializar um array com valores.
.packed-switch Usada para criar uma tabela de salto compactada para a instrução switch.
.sparse-switch Usada para criar uma tabela de salto esparsa para a instrução switch.
.catch Usada para declarar um bloco catch em um método.
.catchall Usada para declarar um bloco catch-all em um método.
.parameter Usada para especificar os parâmetros de um método.
.prologue Usada para marcar o início de um método.
.epilogue Usada para marcar o final de um método.
.source Usada para especificar o nome do arquivo de origem.
.end Usada para marcar o final de uma seção ou bloco.
.restart Usada para reiniciar a contagem dos registradores em um método.
